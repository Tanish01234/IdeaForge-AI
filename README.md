### Prerequisites
- Node.js 18+
- npm or yarn
- API keys (see below)

### 1. Clone & Install

```bash
# Install frontend deps
npm install

# Install backend deps
cd backend && npm install
```

### 2. Configure API Keys

```bash
cp .env.example .env
```

Open `.env` and fill in your keys:

| Variable | Where to Get It |
|---|---|
| `OPENAI_API_KEY` | [platform.openai.com](https://platform.openai.com/api-keys) |
| `STABILITY_API_KEY` | [platform.stability.ai](https://platform.stability.ai/account/keys) |
| `CLOUDINARY_CLOUD_NAME` | [cloudinary.com/console](https://cloudinary.com/console) |
| `CLOUDINARY_API_KEY` | [cloudinary.com/console](https://cloudinary.com/console) |
| `CLOUDINARY_API_SECRET` | [cloudinary.com/console](https://cloudinary.com/console) |
| `REMOVE_BG_API_KEY` | [remove.bg/api](https://www.remove.bg/api) |
| `REPLICATE_API_TOKEN` | [replicate.com/account](https://replicate.com/account/api-tokens) |

### 3. Run Development Servers

Open **two terminals**:

```bash
# Terminal 1 вЂ“ Backend (port 8000)
```
